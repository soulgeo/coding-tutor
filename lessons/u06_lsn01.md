## Health Metrics Tool

In this project, you'll build a BMI (Body Mass Index) category calculator. This requires combining math with a series of logical checks using `if` and `elif`.

BMI is calculated as: `weight / (height ** 2)`.

---

### Your Task
1. Calculate the `bmi` using the formula: **weight divided by (height times height)**.
2. Write an `if/elif` structure to return the correct category string:
   - If BMI is **less than 18.5**, return `"Underweight"`.
   - If BMI is **less than 25**, return `"Normal"`.
   - If BMI is **less than 30**, return `"Overweight"`.
   - For any other value, return `"Obese"`.
3. Click **Submit**.
