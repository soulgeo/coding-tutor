## Comparing Values

We often create Booleans by comparing two values. Python uses special symbols for this:

- `==` (Equal to) - *Note: Use two equals signs!*
- `!=` (Not equal to)
- `>` (Greater than)
- `<` (Less than)
- `>=` (Greater or equal)
- `<=` (Less or equal)

```python
print(5 > 3)  # Prints True
print(10 == 3) # Prints False
```
