## Variable Math

Variables aren't just for storing values; you can do math with them too!

### Updating a Variable
You can use a variable's current value to calculate its new value:
```python
score = 10
score = score + 5 # score is now 15
```

Python reads the right side **first** (`score + 5`), calculates it using the old value, and then saves the result back into the name on the left.
