## Hello World

In the last lesson, you saw the line `print("I am a programmer!")`. That line is our very first **function**.

### What is a Function?
Think of a function like a "command" or a "special tool." It's a piece of pre-written code that does a specific job. The `print()` function has one job: to take whatever you give it and show it in the Console.

When you want a function to do something, you put the information it needs inside parentheses. In programming, we call the stuff inside the parentheses an **argument**. If you leave the parentheses empty, like `print()`, Python will just print a blank line.

To tell Python that you are giving it a message (and not another command), you must wrap your text in quotes (" ").
