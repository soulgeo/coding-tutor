## Loops and Lists

In the last unit, you used `for i in range(5)` to count through numbers. When working with collections, the loop variable doesn't count—it actually **becomes** the data inside the list.

Look at this example with a list of strings:
```python
fruits = ["Apple", "Banana", "Cherry"]
for fruit in fruits:
    print("I like " + fruit)
```

In this loop, the variable `fruit` is `"Apple"` on the first turn, then `"Banana"`, and finally `"Cherry"`. Python handles all the heavy lifting of moving to the next item and stopping at the end.
