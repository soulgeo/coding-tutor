## The "while" Loop

Loops let you run the same block of code multiple times. A `while` loop continues as long as a condition is **True**.

### The Counter
Notice the line `count = count + 1`. This is critical! If we don't update the variable, the condition `count <= 3` will stay True forever, and the program will never stop (this is called an "Infinite Loop").
