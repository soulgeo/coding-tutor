## Your First Program

Welcome to the world of programming! Python is a high-level, easy-to-read language used for everything from web development to Artificial Intelligence.

### The `print()` Function
In Python, we use the `print()` function to display information on the screen. To tell Python to treat something as text, we wrap it in **quotes** (`" "`).

#### Example:
```python
print("Welcome to the course!")
```

### Your Mission
Your first task is a classic tradition in programming. You need to write a program that outputs the phrase "Hello world" to the console.

**Instructions:**
1. Use the `print()` function.
2. Inside the parentheses, type `"Hello world"` (make sure to include the quotes!).
3. Click the **Run** button to check your result.

> **Hint:** > Capitalization and punctuation inside the quotes matter! Make sure it matches exactly: `"Hello world"`.
