## The Final Challenge: Student Grade Analyzer

You have reached the ultimate test! To pass this lesson, you must write a program from scratch that uses variables, lists, functions, and logic.

### Your Goal
Write a program that analyzes a student's grades and provides a summary.

1.  **The Data**: Create a list called `grades` containing three numbers: `100`, `80`, and `60`.
2.  **The Logic**: Define a function named `evaluate(ls)` that:
    - Calculates the **average** of the list (Sum of items divided by the number of items).
    - **Prints** the average.
    - If the average is `80` or higher, **prints** the word `"Excellent"`.
    - Otherwise, **prints** the word `"Good"`.
3.  **The Execution**: Call your `evaluate` function using the `grades` list you created.
