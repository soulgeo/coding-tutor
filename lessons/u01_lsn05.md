## The Invisible Note

As your programs get bigger, you'll want to leave notes for yourself (or other programmers) to explain what the code is doing.

### Comments
In Python, any line starting with a hash symbol (`#`) is a **comment**. Python completely ignores these lines when running your code.

Programmers use comments to:
- Explain complex logic.
- Leave "To-Do" lists.
- Temporarily disable a line of code for testing.
