## Numbers & Math

Python is an amazing calculator. It can handle simple math and complex equations instantly.

### Basic Operators
- `+` (Addition)
- `-` (Subtraction)
- `*` (Multiplication)
- `/` (Division)

### The Comma Trick
When you want to show a message and a number at the same time, you can separate them with a **comma** in the `print()` function. Python will automatically add a space between them for you!

```python
print("Total:", 5 + 5) # Prints: Total: 10
```
