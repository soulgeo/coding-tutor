## Variable Scope

Where you create a variable determines where you can use it. This is called **Scope**.

1. **Local Scope**: A variable created *inside* a function only exists there.
2. **Global Scope**: A variable created *outside* a function can be seen everywhere.

Notice in the example how the `x` inside the function doesn't change the `x` outside!
