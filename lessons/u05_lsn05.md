## Variable Scope

Where you create a variable determines where you can use it. This is called **Scope**.

1.  **Local Scope**: A variable created **inside** a function only exists there. It is "private" to that function.
2.  **Global Scope**: A variable created **outside** a function can be seen by everyone, but functions usually prefer their own local variables if the names are the same.

In the example, we have a global `multiplier`. If you create a local one inside the function, Python will use that one first!
