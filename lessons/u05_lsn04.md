## Multiple Parameters

Functions become truly powerful when they can process multiple pieces of information at once. To give a function more than one input, you simply list the parameter names inside the parentheses, separated by commas.

### How it Works
When you define a function with multiple parameters, you are setting up several "placeholders" that will be filled when the function is called.

```python
def describe_pet(name, animal_type):
    # Remember the 'Comma Trick' from Unit 1? 
    # It works with parameters too!
    print("Pet:", name, animal_type)

describe_pet("Luna", "Cat")
```

**Order Matters!** Python matches your inputs (arguments) to the parameters in the order you provide them. In the example above, `"Luna"` goes into `name` and `"Cat"` goes into `animal_type`.
