## Default Values

You can make parameters optional by giving them a **default value**. This is useful for settings that usually stay the same, but can be changed if needed.

```python
def greet(name="Friend"):
    print("Hello " + name)
```

If you call `greet()`, it uses "Friend". If you call `greet("Alice")`, it uses "Alice".
