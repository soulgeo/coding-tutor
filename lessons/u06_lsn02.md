## Social Media Handle Generator

Processing text is a huge part of programming. To do this, Python provides built-in tools called **String Methods**.

### The .upper() Method
One of the most common methods is `.upper()`. You use it by putting a dot after your string or variable. It returns a new version of the text in all capital letters.

```python
name = "alice"
print(name.upper()) # Prints "ALICE"
```

### Concatenation
Remember that you can use the `+` symbol to "glue" strings together. For example, `"@" + "USER"` becomes `"@USER"`.
