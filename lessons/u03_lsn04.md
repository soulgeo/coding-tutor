## The range() Function

If you want to run a `for` loop a specific number of times, use `range()`.

- `range(5)` -> 0 to 4 (5 items)
- `range(1, 6)` -> 1 to 5
- `range(0, 10, 2)` -> 0, 2, 4, 6, 8 (count by 2)

Notice that Python usually starts counting at **0** and stops **before** the final number.
