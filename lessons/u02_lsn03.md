## The "if" Statement

The `if` statement tells Python: "Only run this code **if** the condition is True."

### Indentation Matters!
In Python, we use **indentation** (4 spaces or a Tab) to show which code belongs inside the `if` statement.

```python
if score > 50:
    print("You passed!") # This is inside the if
print("End of program")  # This is outside (runs no matter what)
```

Don't forget the **colon** (`:`) at the end of the `if` line!
