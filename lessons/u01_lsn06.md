## What is a Variable?

In programming, we need to save data so we can use it later. We do this using a **Variable**.

Think of a variable like a labeled box. You can put something inside the box, give it a name, and then pull it out whenever you need it.

### The Assignment Operator
To create a variable, we use the equals sign (`=`).
```python
name = "Alice"
score = 100
```
This tells Python: "Take the value on the **right** and store it in the name on the **left**."
