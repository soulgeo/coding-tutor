## Password Validator

Security is a major part of programming. Most websites have rules about how long a password must be to keep accounts safe.

### Checking Length
You already know how to use `len()` to find the length of a list. In Python, `len()` works exactly the same way for strings!

```python
name = "Alice"
print(len(name)) # Prints 5
```
