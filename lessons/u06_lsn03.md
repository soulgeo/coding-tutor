## The Data Cleaner

In the real world, data is often "messy." It might contain errors, negative numbers that shouldn't be there, or "empty" values. Programmers use **filtering** to clean this data before using it.

### The Filtering Pattern
To filter a list, you follow three steps:
1. Create a new, empty list (like `cleaned = []`).
2. Loop through the original list.
3. Use an `if` statement to decide if the current item should be `.append()`ed to the new list.
