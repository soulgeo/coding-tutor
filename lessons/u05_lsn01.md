## Defining a Function

A **Function** is a reusable block of code that does a specific job. Instead of writing the same code over and over, you can put it in a function and "call" it whenever you need it.

To create a function, we use the `def` keyword (short for "define").

```python
def my_function():
    print("Doing work...")
```
