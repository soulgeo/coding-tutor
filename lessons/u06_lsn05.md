## User Access Control

In larger applications, you'll often have different levels of access. You can use multiple lists to keep track of who is who.

### Multi-List Logic
You can use the `in` keyword on different lists to determine a user's role:
```python
if name in team_a:
    # ...
elif name in team_b:
    # ...
```
