## The Secret of Quotes

In Python, there is a big difference between a **command** and a **message**. 

### What is a String?
In programming, we call text a **String**. Think of it like a "string of characters" tied together. To tell Python that something is a message and not a command, you must wrap it in quotes.

- `print("Hello")` -> Python sees a message: "Hello".
- `print(Hello)` -> Python looks for a command or a variable named `Hello`. Since it doesn't find one, it will crash!

Python is flexible! You can use double quotes (`" "`) or single quotes (`' '`). Both work exactly the same way:
- `print("Hi!")`
- `print('Hi!')`

The only rule is that you must **start and end with the same type**. You can't do `"Hi!'`.
