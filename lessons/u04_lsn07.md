## Unit 4 Recap

Lists allow you to handle large amounts of data:
- Use `[]` to create a list.
- Use `[index]` to get an item (starting at 0).
- Use `.append()` and `.pop()` to change the list.
- Use `len()` to get the size.
- Use `for` to loop through items.
- Use `in` to search.
