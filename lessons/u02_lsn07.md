## Unit 2 Recap

You've learned how to make your programs smart!
- **Booleans** are True or False values.
- **Comparison operators** (`==`, `>`, `<`, etc.) create Booleans.
- **if / elif / else** control which code runs.
- **and / or / not** let you create complex logic.
